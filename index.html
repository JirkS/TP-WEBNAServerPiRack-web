WEBNAServer Pi rack

Components

The heart of the tech project lies in the powerful Raspberry Pi 4 Type B, a versatile single-board computer that provides the brains for endeavour. Enhancing the user interface is a small OLED RGB display, adding a vibrant visual element to the project. To ensure optimal performance and longevity, a dedicated fan has been integrated to keep the Raspberry Pi cool under varying workloads.

Case Rack

The physical housing for the tech setup is a product of modern design and manufacturing. Utilizing Rhinoceros 7 for precise 3D modeling, the case rack was printed on a Prusa MK4 3D printer, demonstrating the fusion of digital design and physical production. The assembly of the components into this custom-designed case rack ensures a sleek and functional enclosure for technological masterpiece.

Used Softwares

A robust online presence requires meticulous setup and configuration. Achieving this involves setting up a public IP address through my home router, installing and configuring the Nginx web server with the public IP address, purchasing a domain name, configuring DNS records in control panel of domain, and securing the website with an HTTPS certificate. The latter is implemented using Certbot by Let's Encrypt, enhancing the security and trustworthiness of the webpage.

Web

The user interface of the project comes to life through the development of a dynamic webpage using HTML and CSS. These foundational web technologies provide the structure and styling necessary for an engaging and visually appealing user experience. The integration of these elements showcases commitment to user-friendly design in the digital realm.

Display with Stats

Elevating the user experience to new heights, a dedicated display has been seamlessly integrated, providing real-time system statistics. Crafted with precision in Python, the script empowers the display to showcase essential metrics such as CPU usage, RAM statistics, and DISK performance. This dynamic presentation not only offers valuable insights into the Raspberry Pi's operations but also enhances user engagement. Furthermore, the public IP address is prominently featured, completing the comprehensive display and underscoring the versatility and functionality of Python-powered display of tech project.
